# Descripción general
Este proyecto trata sobre una plataforma web para gestionar los gastos personales. La idea principal es que los usuarios puedan registrar gastos de forma manual o cargando una foto de un ticket comercial.   
A partir de la información cargada el usuario podrá visualizar estadísticas y gráficos de consumo. Este sistema además permitirá que un asesor financiero pueda analizar los patrones de consumo de los usuarios y brindarle recomendaciones según el usuario los soliciten.  
El agregado de IA permitirá extraer de forma automática el monto, comercio y fecha del ticket.
# Roles de usuario
Usuario: Gestión de sus gastos, carga de tickets, visualizar estadísticas personales.  
Asesor: Acceso al patrón de consumo de los usuarios y generar recomendaciones
